# Подключение к Backend API

## Быстрый старт

1. **Запустите backend сервер:**
   ```bash
   cd backend
   python run.py
   ```

2. **Проверьте, что сервер работает:**
   Откройте в браузере: http://localhost:8000/docs

3. **Настройте URL API (если нужно):**
   Файл `.env` уже создан с правильным URL:
   ```
   REACT_APP_API_URL=http://localhost:8000
   ```

4. **Запустите React приложение:**
   ```bash
   cd miniapp/react-app
   npm start
   ```

5. **Проверьте консоль браузера:**
   - Откройте DevTools (F12)
   - Должны быть сообщения об успешном подключении к API

## Готово!

Теперь фронтенд подключен к backend через HTTP. Все данные загружаются из базы данных.

## Если не работает

1. Убедитесь, что backend запущен на порту 8000
2. Проверьте файл `.env` - URL должен быть `http://localhost:8000`
3. Перезапустите React приложение после изменения `.env`
4. Проверьте консоль браузера на наличие ошибок

Подробнее см. `CONNECTION_GUIDE.md` в корне проекта.
